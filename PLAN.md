# Getting Started with Quiz Management System App

- Architecture : molecular architecture
- Tech stacks : React.js, Typescript, HTML, modular CSS, Material UI
- Scope : User of taking quiz tests and Admit how manages the quizzes
- Admin Features : Create a quiz, Add questions to quiz and manage them (delete, modify, add)
- Public User Features : Select quiz to attend, select answers for questions and submit them and see the result

# How to start the App

- Clone the repo from the git
  ```bash
      git clone https://github.com/suneelfm/quiz-management-system.git
  ```
- Install all the app dependencies
  ```bash
      npm i
  ```
- Start the app server
  ```bash
      npm start
  ```
